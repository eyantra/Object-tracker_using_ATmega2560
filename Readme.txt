Object Tracker 
------------------------

Hardware Requirements
-----------------------------------
+ AVR Robo
+ Xbee Module
+ Motors
+ Motor Drivers
+ Camera

Software Requirements
--------------------------------------
+ AVR Studio
+ C compiler
+ Matlab

Introduction:
------------------
	
	 To track a moving target using a robust visual tracking algorithm and shoot the laser mounted on the robot such that it points the moving target.
